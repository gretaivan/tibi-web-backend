# tibi-web
Launch teaser website to collect people interest
 
## BACKEND

### First time use 
+ fork and clone this repo
+ open a root directory in a terminal
+ ```npm install``` to install dependencies

### Usage 
+ npm start to launch server
+ npm test to start test
